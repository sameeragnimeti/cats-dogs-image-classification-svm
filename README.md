# cats-dogs-image-classification-svm
Image classification of cats and dogs using Support Vector Machine (SVM) with Python and OpenCV.# Cats vs Dogs Image Classification using SVM

This project implements a Support Vector Machine (SVM) model to classify images of cats and dogs using the Kaggle Dogs vs Cats dataset.

## Technologies Used
- Python
- OpenCV
- NumPy
- Scikit-Learn
- Jupyter Notebook

## Workflow
1. Load image dataset
2. Resize and preprocess images
3. Convert images to feature vectors
4. Train SVM classifier
5. Evaluate model accuracy

## Dataset
Kaggle Dogs vs Cats Dataset

## Output
The model predicts whether an image contains a cat or a dog.
